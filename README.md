# FruitRun
## Project Overview & Story
**Fruit Run** is an interactive 3D obstacle runner built from scratch using Three.js and custom WebGL/GLSL shading concepts, developed for UCLA's CS 174A course in spring quarter 2026.

**Story:** You play as a girl who has just gathered fruit to bring home to her family. However, the path home has obstacles that you must avoid. Each time you run into an obstacle, you drop fruit. Your goal is to reach the end of the path having hit the fewest obstacles so that you have more fruit to feed your family.
