# FruitRun
## Project Overview & Story
**Fruit Run 3D** is an interactive 3D obstacle runner built from scratch using Three.js and custom WebGL/GLSL shading concepts.
## How to Run 
### Option A: Using Vite 
1. Open your terminal in the root directory of the project.
2. Install the necessary dependencies (excluding `node_modules` from tracking):
    ```bash
    npm install
3. To run project
    ```
    npx vite
4. Open the provided URL in your browser
